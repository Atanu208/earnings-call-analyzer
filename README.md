
# AI Earnings Call Analyzer

Analyze earnings call transcripts using AI-powered insights such as:

- SWOT Analysis
- Moat Detection
- Tailwinds
- Headwinds
- OLAELEC Sample Included


## 🚀 To Run

### 1. Install dependencies (one time)

npm install

### 2. Create your `.env` file in the root folder

Edit `.env` and paste your free API key from:

[https://console.groq.com/keys](https://console.groq.com/keys)

Sample:

```env
GROQ_API_KEY=your_api_key_here
PORT=3000
```

### 3. Start the project

```bash
npm start
```

Open in browser:

```bash
http://localhost:3000
```

---

## 📌 Notes

* Make sure `.env` is in the root folder
* Default port is `3000`

---

Upload or paste an earnings call transcript and get instant AI analysis with strategic insights.
