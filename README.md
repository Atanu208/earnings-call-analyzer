To run:
1. install dependencies (one time)
npm install

2. create your .env file in the root folder
edit .env → paste your free key from console.groq.com/keys and PORT number
sample:
GROQ_API_KEY=gsk_0HZKi567654gnghfdffhjhhjgfre565trthghgrWdIF8sUMym
PORT=3000

3. start
npm start
→ http://localhost:3000
